# Design and Analysis of Algorithms (DAA) Laboratory

This repository contains the implementation of various algorithms covered in the **Design and Analysis of Algorithms** Lab. The goal of this lab is to understand the practical aspects of algorithm design, their time/space complexities, and various algorithmic strategies.

---

## 🚀 Lab Overview
The experiments are implemented using **[Mention Language, e.g., C++ / Java / Python]** and focus on analyzing the performance of algorithms under different input sizes.

### 📚 Algorithmic Strategies Covered
* **Divide and Conquer:** Breaking problems into smaller sub-problems.
* **Greedy Method:** Making locally optimal choices at each stage.
* **Dynamic Programming:** Solving problems by combining results of sub-problems.
* **Backtracking:** Exploring all possible solutions.
* **Graph Algorithms:** Traversals and shortest path calculations.
